# Loan Approval Prediction using Machine Learning

## 📌 Overview
This project predicts whether a loan application will be approved based on applicant information using machine learning classification algorithms.

## 📊 Algorithms Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest

## 📚 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## 📈 Workflow
1. Data Loading
2. Data Exploration
3. Exploratory Data Analysis (EDA)
4. Data Preprocessing
5. Model Training
6. Model Evaluation
7. Model Comparison

## 🏆 Best Performing Model
The notebook compares the performance of all implemented classification models based on accuracy.
